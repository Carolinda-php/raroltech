3. Manipulação de Arquivos com Log Rotativo

• Para sistemas que gravam muitos logs, o gerenciamento de arquivos pode ser feito por meio de logs rotativos, onde um novo arquivo é criado quando o anterior atinge um certo limite. Logs rotativos são usados para evitar que arquivos de log fiquem grandes demais. Quando o arquivo atual atinge o tamanho máximo, é criado um novo arquivo de log.
